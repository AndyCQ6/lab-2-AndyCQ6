1. ¿Cuáles son los resultados de las siguientes expresiones?

   | 13 % 9     |     4     |
   | 13 / 2 % 2 |     25    |

2. ¿Qué sucede cuando el usuario ingresa un número que tiene menos de cinco dígitos? ¿Por qué? ¿Cuál es la salida cuando se ingresa 1763?
